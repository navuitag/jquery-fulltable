jQuery fullTable
=========

Easy Inline Editor for HTML tables - jQuery fullTable

## Installation

  `npm install @navuitag/jquery-fulltable`

## Usage

    var numFormatter = require('@navuitag/jquery-fulltable');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`


## Tests
